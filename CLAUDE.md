# ประมาณการโปรเจค (Profit Estimation)

เว็บแอปหน้าเดียวสำหรับบันทึกข้อมูลหลายแผนก และสรุปกำไร(ขาดทุน)หน่วยงาน (P&L)

## โครงสร้าง
- ทั้งแอปอยู่ใน `index.html` ไฟล์เดียว (~1,600 บรรทัด: HTML + CSS + JS รวมกัน)
- UI ภาษาไทย · ฟอนต์ IBM Plex Sans Thai / IBM Plex Mono
- 6 แผนก: `sales`, `cost`, `ap`, `prelim`, `ops`, `gl` + แท็บ audit (Admin)
- ตาราง P&L คำนวณอัตโนมัติ, ระบบ lock แผนก, audit log, ปฏิทินเลือกช่วงวันที่

## การเก็บข้อมูล
- localStorage (ออฟไลน์) + ซิงค์ขึ้น Google Apps Script Web App → Google Sheet
- `API_URL` (ราว ๆ บรรทัด 451) ชี้ไป Web App ที่ deploy แล้ว
- โค้ด `.gs` อยู่ฝั่ง Google **ไม่ได้อยู่ใน repo นี้** — ในไฟล์มีแค่ URL
- ⚠️ ข้อมูลตอนนี้เป็น **dummy ทั้งหมด** (ดูฟังก์ชัน `seed()`) — กำลังโฟกัส UI ก่อน แล้วค่อยทำความปลอดภัย + ใส่ข้อมูลจริง

## วิธีทำงาน (solo)
- commit + push เข้า `main` ตรง ๆ ได้ (ไม่ต้องเปิด PR) · commit message ภาษาไทยได้
- **ก่อนสลับเครื่อง:** `git push` ทุกครั้ง · **เริ่มงานบนเครื่องใหม่:** `git pull` ก่อนเสมอ

## Preview (local)
- เปิด `index.html` ตรง ๆ ได้ แต่ฟีเจอร์ Apps Script ต้องรันผ่าน dev server
- ⚠️ บน macOS ถ้าโปรเจคอยู่ใต้ `~/Desktop` ตัว dev server จะถูก TCC บล็อก → เสิร์ฟ "สำเนา" จาก `/tmp` แทน (ถ้า clone ไปไว้นอก Desktop จะเสิร์ฟโฟลเดอร์ตรง ๆ ได้)

## งานที่กำลังทำ + เอกสาร
